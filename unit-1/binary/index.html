Binary


What is binary?

Binary is a base-2 number system which only uses ones and zeros.

How does it work?

Binary consists of bits which each contain a single value, a one or a zero. these bits cannot consist of any other value. The first bit in binary is the ones place and with every bit the value doubles. For example: 1, 2, 4, etc. If the bit in that place is 1 then it should be added. For example: 101 = 5.
